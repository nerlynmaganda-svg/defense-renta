About the Project

What is RENTA?

RENTA is a web-based online apartment management system developed to address the inefficiencies and lack of transparency in the apartment rental process in Lipa City, Batangas. The system integrates immersive 360° virtual tour technology through Panoee, enabling tenants to explore properties remotely before visiting in person.

  • ✓Browse and search apartments with advanced filters
  • ✓Immersive 360° virtual tours via Panoee integration
  • ✓Direct messaging and inquiry system
  • ✓Online reservation and payment processing
  • ✓Google Maps integration for location accuracy
  • ✓Role-based dashboards: Admin, Landlord, Tenant
  • ✓Real-time notifications and alerts
  • ✓Mobile browser compatible — no app download needed
🏢
Panoee Integration
Apartment owners upload panoramic images and the system automatically generates interactive 360° virtual tours — allowing tenants to walk through properties online as if they were physically there.

⚠ System Limitations

The system is not available as a native mobile app (no offline access, no home-screen pinning, no fingerprint reader). Virtual tour performance depends on Panoee's external service. Features like map location, notifications, reservations, and payments require a stable internet connection.
